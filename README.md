Assignment Requirements:  
1.POST /users  
Accept name and email in JSON  
Validate both fields  
Store the data in an in-memory slice (no database required).  
  
2.GET /users  
Return the list of all users that were added through POST  
You may use the Gin framework or the standard library.  

I have used Go Version 1.24.5  
You can start the server simply using the command "go run main.go" inside the folder.
